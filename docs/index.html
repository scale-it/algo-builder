Framework to automate development of Algorand Assets and Smart Contracts

Algo Builder is an trustworthy framework for Algorand dapps (Decentralized Applications). Its main goal is to make shipping Algorand applications simple, efficient, and scalable. Available packages:

  • algob: a framework of the Algo Builder Suite
  • types/algosdk: TypeScript typings for algosdk-js
  • runtime: Light Algorand runtime and TEAL interpreter

Example Code

Check this for the original implementaion (using SDK takes approx. ~450 LOC). Below snippet demonstrates the same using algob in 15x less code: 

/*
    Transfer some algo from master to account
    Create "gold" Algorand Standard Asset (ASA). // definition in `/assets`
    optIn to ASA "gold"
*/
import { executeTransaction, balanceOf } from '@algorand-builder/algob';
import { rtypes } from '@algorand-builder/runtime';

async function run (runtimeEnv, deployer) {
  const masterAccount `=` deployer.accountsByName.get('master-account');
  const goldOwner `=` deployer.accountsByName.get('alice');
  const bob `=` deployer.accountsByName.get('bob');

  // transfer ALGO
  await executeTransaction(deployer, {
    type: types.TransactionType.TransferAlgo,
    sign: types.SignType.SecretKey,
    fromAccount: masterAccount,
    toAccountAddr: goldOwner.addr,
    amountMicroAlgos: 5e6, // 5 ALGO
    payFlags: { note: 'funding account' }
  })

  // deploy ASA
  await deployer.deployASA('gold', { creator: goldOwner, totalFee: 1001 });

  // optIn to ASA (by bob-account)
  await deployer.optInAcountToASA('gold', 'bob', {});
}
